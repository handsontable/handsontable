DEV-1800: Published .d.ts downleveled to TS 5.1+ compatible

A new scripts/downlevel-dts.mjs post-processes all tmp/**/*.d.ts files after the type build. It replaces TypeScript lib types introduced after TS 5.1 with TS-5.1-compatible equivalents. The dev compiler remains on TS 6 — only the published output is downleveled.

Build pipeline

1. build:types — tsc emits tmp/**/*.d.ts (may contain newer lib types)
2. downlevel:types — rewrites newer-than-5.1 lib types (ArrayIterator → IterableIterator, WeakKey → object, etc.)
3. test:types — verifies published declarations compile clean under TS 5.1.6
