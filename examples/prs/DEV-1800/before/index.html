DEV-1800: TypeScript lib types leak into published .d.ts

Handsontable is built with TypeScript 6.x, which emits newer lib types into generated .d.ts files. Any consumer using TypeScript ≤ 5.1 (e.g. Angular 16's max supported version) gets TS2304: Cannot find name 'ArrayIterator' errors.

selection/range.d.ts~134ArrayIterator<CellRange>
