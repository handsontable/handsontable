DEV-1722: afterFormulasValuesUpdate hook – time cell values

Setup: Column A holds a plain number (hours, 0–23). Column B has the formula =A1/24 with type:'time', so HyperFormula returns a day-fraction and HOT should display a formatted time string.

How to test: Click a cell in column A, type a number (e.g. 15), press Enter.

Before fix (v17.0.1): newValue in the hook is a raw HF day-fraction (e.g. 0.625).
After fix (v17.1.0): newValue is a formatted time string (e.g. "15:00").
