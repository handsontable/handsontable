The HyperFormula Dashboard turns any CSV, JSON, or Excel file into an editable spreadsheet with live KPIs, tables, and charts. Filter columns or edit cells and every widget recalculates -- powered by Handsontable for the grid and HyperFormula for 400+ Excel and Google Sheets compatible functions.

[Open the HyperFormula Dashboard](https://hyperformula-dashboard.lovable.app)


## Upload data and build widgets

Drop in a **CSV**, **JSON**, or **Excel** file, or pick one of the bundled samples -- Inventory, Transactions, Support Tickets, or Employees. Your data lands in an editable spreadsheet with column filters and cell editing.

From there you add widgets to the dashboard. Four widget types are available: **KPI** (Key Performance Indicator) for single-number metrics, **Table** for grouped summaries, **Pie** for share-of-total breakdowns, and **Bar** for category comparisons. Edit or filter the source data, and all the widgets update instantly.

## AI writes formulas, not code

The AI assistant writes the formulas behind each widget in plain spreadsheet syntax -- `SUMIF`, `COUNTIF`, `AVERAGEIFS`, and the rest of the 400+ functions HyperFormula supports. You see the formula, you can edit it, and a business user can read it without learning a new language.

Because the output is a real spreadsheet formula, the same logic works if you paste it into Excel or Google Sheets. That makes the calculations explainable, auditable, and portable across tools your team already uses.
