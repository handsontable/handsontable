Dropdown

This example shows the usage of the Dropdown feature.

Dropdown is based on Autocomplete cell type. All options used by autocomplete cell type apply to dropdown as well. The only difference is that filter option is always set to false and strict is always set to true. Therefore you can think of dropdown as a searchable <select>.
