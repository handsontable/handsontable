DEV-1857: Middle mouse button autoscroll

Issue: #2722 — pressing the scroll wheel over the grid starts no autoscroll on Windows/Linux

Switch to: BEFORE FIX v17.1.0 AFTER FIX patched
Bug: HOT v17.1.0 calls event.preventDefault() on every mousedown, including middle-button (button 1). The browser's native autoscroll requires the mousedown default action to be preserved — canceling it suppresses the panning cursor entirely.

How to test (Windows / Linux):
  1. Middle-click (press the scroll wheel) on any grid cell below.
  2. No panning cursor appears — autoscroll is broken.
  3. The status panel shows defaultPrevented: true.
  4. Compare with the plain scroll container at the bottom — middle-click works there.
