⚠ Broken State

Problems in the "before" state

  • 🚫
    Walkontable excluded from main TypeScript type-check handsontable/tsconfig.json had "src/3rdparty/**" in the exclude array. The tsc --noEmit / CI typecheck task never checked Walkontable's 83 .ts files (~17.6k lines). Any type regression in Walkontable would silently pass CI.
