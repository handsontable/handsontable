Bug: TypeError: Cannot read property 'isHorizontallyScrollableByWindow' of undefined
Root cause: The StretchColumns ResizeObserver callback used this.hot?.view.isHorizontallyScrollableByWindow() — optional chaining on hot but NOT on view. If view is undefined (race condition during init/teardown), a TypeError is thrown.

This demo patches the ResizeObserver callback to reproduce the pre-fix behavior.
