PR #12211 - nestedHeaders + columnSorting crashes when disableVisualSelection: 'current'

Build: Released v17.0.1 BUG PR #12211 Fix FIXED
BUG present (v17.0.1). When disableVisualSelection: 'current' is set, the focus highlight's visualCellRange is null. The NestedHeaders plugin's onBeforeHighlightingColumnHeader handler tries to access focusHighlight.visualCellRange.highlight.col without a null check, throwing a TypeError: Cannot read properties of null on every sort click.
How to observe the bug:
  1. Click "Click column header to sort" — this simulates clicking the "Name" column sort arrow.
  2. A TypeError: Cannot read properties of null (reading 'highlight') is thrown.
  3. Sorting does not complete — the grid stays unsorted.
  4. Compare with the Fix page where sorting works correctly.
