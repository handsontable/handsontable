DataProvider CRUD — PR #12172

Version: Released v17.0.1 PR Build local
Bug — no CRUD callbacks in v17.0.1: The dataProvider plugin does not exist in v17.0.1. Editing a cell updates the local data store immediately with no server notification. Right-clicking to insert or remove rows also has no server-side callback. The "Server log" below stays empty no matter what you do — the grid has no mechanism to report changes to a backend.

Steps to reproduce: 1. Edit any cell (double-click or press F2). Confirm and press Enter. 2. Right-click a row header to insert or remove a row. 3. Observe: the Server log remains empty — no onRowsCreate, onRowsUpdate, or onRowsRemove callbacks exist.
