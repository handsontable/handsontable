Build: Released v17.0.1 BUG PR #12129 Fix FIXED PR #12152 Fix FIXED
FIX applied (PR #12129 — local build). The nestedHeaders state manager now reads the rowspan property from header config objects and a new applyRowspanToMatrix post-processor marks covered cells as placeholders. Walkontable renders a rowspan attribute on the resulting <th>. A new isBottomMostColumnHeader(TH) DOM helper correctly identifies rowspanned headers as bottom-most, enabling sort indicators, dropdown buttons, and filter badges on them. 
// stateManager/settingsNormalizer.js — rowspan support added:
// Maintains a rowspanCoverageMap so covered source columns are skipped
// and { label, rowspan } is mapped to the correct renderable column.

// helpers/dom/element.js — new helper:
export function isBottomMostColumnHeader(TH) {
  const headerLevel = headerRows.indexOf(headerRow);
  const rowspan = Number.parseInt(TH.getAttribute('rowspan'), 10) || 1;
  return headerLevel + rowspan >= headerRows.length;
}
