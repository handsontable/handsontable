PR #12299 - Notification plugin (toast messages)

Version: Released v17.0.1 PR Build local
Fix: The new notification plugin is enabled via notification: true. Use hot.getPlugin('notification').showMessage({...}) to display toast notifications. Click the buttons below to trigger toasts of each variant, a persistent toast with an action button, or hide all active toasts.

Keyboard: F6 moves focus to the notification region. Escape returns focus to the grid.
