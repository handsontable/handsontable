Server-side sorting via dataProvider — PR #12160

Version: Released v17.0.1 PR Build local
Fix — server-side sorting via dataProvider: With PR #12160, enabling both dataProvider and columnSorting delegates sorting to the server. Clicking a column header triggers fetchRows with a sort: { prop, order } field in the query params. The grid displays the pre-sorted rows returned by the server. Client-side reordering is disabled. Note: multiColumnSorting is automatically disabled with a console warning when dataProvider is set.

Click a column header to sort - the server receives the sort parameter and returns pre-sorted data.
