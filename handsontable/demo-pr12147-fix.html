Fix - DataProvider plugin is available in this PR build.
The dataProvider option enables server-side row loading via fetchRows(queryParams, { signal }). The mock server below holds 15 cars. Page size is 5, giving 3 pages. Each fetchRows call is logged in the Server log below.

How to test: 1. The grid loads page 1 automatically on init. 2. Click Fetch page 2 to trigger a manual fetchData({ page: 2 }) call via hot.getPlugin('dataProvider').fetchData(). 3. Watch the Server log for the query params sent to the mock server. 4. Check Total rows from server and Current page metrics. 5. Open the console and run hot.getPlugin('dataProvider').getQueryParameters() to inspect current params.
