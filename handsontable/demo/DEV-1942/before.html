DEV-1942: Skip eager full-grid cell-meta materialization on load

Version: BEFORE fix buggy AFTER fix fixed
Bug: On init, getCellMeta() is called for every cell (O(rows×cols)) regardless of whether a sourceDataValidator is configured. This materializes and permanently caches a meta object for every cell, wasting memory and slowing initialization dramatically for large grids.
