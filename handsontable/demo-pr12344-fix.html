PR #12344 - Drag-select during auto-scroll (DragToScroll) NEW FEATURE

Build: Released v17.0.1 BUG PR #12344 Fix FIXED
FIX applied (PR #12344). Drag-select now extends the selection on every auto-scroll tick, matching the behavior of Excel and Google Sheets. On each scroll tick, afterScroll fires, getCellCoordsFromMousePosition maps the last known mouse position to the current edge cell, and selection.setRangeEnd extends the range to that cell.
  • Drag down past the bottom edge → selection grows into newly revealed rows. ✓
  • Drag right past the right edge → selection grows into newly revealed columns. ✓
  • Drag back → selection shrinks naturally. ✓
  • Two new public hooks: beforeOnCellMouseOverOutside and afterOnCellMouseOverOutside. ✓
// FIX — DragToScroll plugin (PR #12344):
// On each auto-scroll tick, afterScroll fires and extends the selection:
#onAfterScroll = () => {
  const lastMousePos = this.#lastMousePosition;
  if (!lastMousePos || !this.hot.selection.isInProgress()) return;

  const coords = getCellCoordsFromMousePosition(this.hot, lastMousePos.x, lastMousePos.y);
  if (coords) {
    this.hot.selection.setRangeEnd(coords);  // extends range to edge cell
  }
};
