PR #12395 - dateFormat TypeScript type for intl-date cell type

Build: Released v17.0.1 BUG PR #12395 Fix FIXED
FIX applied (PR #12395). The dateFormat option in settings.d.ts is now typed as string | Intl.DateTimeFormatOptions. TypeScript users can pass a native Intl.DateTimeFormatOptions object to an intl-date column without any cast. The JSDoc in metaSchema.js already documented both forms; this aligns the type definitions to match. 
// FIX — handsontable/types/settings.d.ts:
interface GridSettings {
  ...
  dateFormat?: string | Intl.DateTimeFormatOptions;
  // ↑ Intl.DateTimeFormatOptions is now valid for intl-date columns
}

// TypeScript regression test added (settings.types.ts):
hot.updateSettings({
  columns: [
    { type: 'intl-date', dateFormat: 'YYYY-MM-DD' },                              // string ✓
    { type: 'intl-date', dateFormat: { year: 'numeric', month: 'long' } },        // object ✓
  ],
});
