Bug - DataProvider plugin is absent in v17.0.1.
The dataProvider option introduced in PR #12147 does not exist in the released v17.0.1 build. Passing it is silently ignored. Data must be loaded statically via the data array — there is no server-side row model, no fetchRows callback, and no pagination driven by the server.

How to verify: Open the browser console and run hot.getPlugin('dataProvider'). You will get undefined (or a "plugin not found" error), confirming the plugin does not exist in this version.
