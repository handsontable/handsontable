DEV-1838: indexSyncer private module leak — fixed

This page demonstrates the fix — the pattern index.* only exports exact entry points.

Version: BEFORE fix BUG AFTER fix FIXED
✅ Fixed: Changed ./plugins/*/+(index)* to ./plugins/*/index.*. The new pattern only matches files named exactly index.js, index.mjs, index.d.ts, etc. The private indexSyncer/ directory is no longer reachable via package imports.

Five patterns changed. Each +(index)* extglob → index.* literal glob. The new pattern requires the filename to be exactly index followed by a dot and extension.
